1. Platform Architecture: Technical Deep Dive

OpenClaw is a Node.js-based gateway that functions as a bridge between messaging apps (WhatsApp, Telegram, etc.) and AI models or tools. It follows a hub-and-spoke architecture where the Gateway acts as the central control plane.

1.1 Core Components

  • Gateway: The always-on HTTP server (default port 18789) that manages sessions, routing, and security.
  • Channels: Integrations for WhatsApp, Telegram, Discord, iMessage, etc.
  • Agents: The "workers" defined by SKILL.md files that execute tasks.
  • Tools: External APIs (ScraperAPI, OpenAI) and local capabilities (file system, shell execution).

1.3 Security Architecture

2. Implementation Guide: From Zero to Production

2.1 Hardware & Software Requirements

  • Hardware: Modern Mac/PC (8GB+ RAM) or VPS (DigitalOcean Droplet, AWS Lightsail).
  • Software: Node.js v22.16+ or v24 LTS.
  • API Keys: OpenAI, Anthropic, and data providers (e.g., ScraperAPI).

2.2 Step-by-Step Setup

Phase 1: Installation

# Install Node.js (MacOS/Homebrew)
brew install node

# Install OpenClaw globally
npm install -g openclaw@latest

# Run the onboarding wizard
openclaw onboard --install-daemon

Phase 2: Configuration

Edit the configuration file located at ~/.openclaw/openclaw.json. Below is an example configuration for WhatsApp and OpenAI:

{
  "channels": {
    "whatsapp": {
      "allowFrom": ["+15555550123"], // Whitelist your number
      "groups": { "*": { "requireMention": true } }
    }
  },
  "providers": {
    "openai": {
      "apiKey": "sk-proj-..."
    }
  },
  "router": {
    "defaultAgent": "chief_of_staff",
    "rules": [
      { "match": "market|comps", "agent": "research" },
      { "match": "call|prospect", "agent": "call_prep" }
    ]
  }
}

Phase 3: Agent Development

Create a directory for your agents. Each agent requires a SKILL.md file.

Example: Research Agent (agents/research/SKILL.md)

# CRE Research Agent
name: cre_research
description: Gathers market intelligence and comparable data
tools:
  - web_browser
  - scraper_api

prompts:
  system: |
    You are a Commercial Real Estate Research Specialist.
    Your goal is to provide accurate, cited market data.
    
    When asked for comps:
    1. Use the 'scraper_api' tool to query public records.
    2. Cross-reference data with recent news.
    3. Output a structured table with:
       - Address
       - Sale Price
       - Cap Rate
       - Date
       - Source URL
